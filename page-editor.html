Comments Form

Created a page for the Feed-back form which contains questions inregards to the user experience when using the website.

Extra Content Page

The extra content page was created based on the theme of the website therefore contains images and descriptions regarding Conferences and Exhibitions along with posters that contains details of upcoming events in the specified field.

The basic layout was discussed within the group members to create cohesion and to maintain the consistency within the webpages.

CSS was used in order to create the layout and the structure for a minimalistic yet modern webpage and the CSS grid layout was used to create the section containing the posters.

Anchor tags are introduced at the intial stage of the webpage to navigate to a specific section within the extra content page.
